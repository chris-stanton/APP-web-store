Software Applications

Guide Spacing Database

Guide Spacing Database is an aide for helping with guide size and spacing when building ice fishing rods. Application has guide size spacing specs from specific manufactures and modified spacings by Chris Stanton. Data output contains blank info like: MFG name, model name, guide sizes/spacing and lure weight recommendations. Spacing and guide size values are current with production or have ben modified by Chris Stanton.

Wrap Designer: Ice Rod Edition

Fishing Rod Wrap Designer is a simulator to show a user their thread color combinations before production starts on the fishing rod. The app gives the user a realistic idea on the design, width and color choices made before production is started on making the fishing rod. The user can pick from real thread MFG colors and change widths to create specific thread patterns. Users can also pick handles and view a scaled down version of the entire fishing rod, that includes: wraps, blank color, and a selected handle styles. Handle styles range from single piece designs to multi-piece designs with tread wraps.
